## 3. 合规认证费用 (USD)

| 认证 | 适用区域 | 单 SKU 费用 | 周期 | 备注 |
|---|---|---|---|---|
| FCC ID (Part 15B+15C) | 美国 | $3,000 - $6,000 | 4-6 周 | 蓝牙/Wi-Fi 必备 |
| FCC SDoC (无无线) | 美国 | $500 - $1,500 | 1-2 周 | 仅 EMC |
| CE RED + EMC + LVD | 欧盟 | $4,000 - $8,000 | 5-8 周 | 含 RF 测试 |
| UKCA | 英国 | $2,000 - $4,000 | 3-5 周 | 脱欧后追加 |
| BQB (蓝牙) | 全球 | $4,000 - $10,000 | 4-8 周 | 含 SIG 会员费 |
| RoHS / REACH | 欧盟 | $500 - $1,500 | 1-2 周 | 物料分析 |
| Telec / MIC | 日本 | $3,000 - $6,000 | 4-6 周 | 日本市场无线 |
| KC | 韩国 | $3,000 - $5,000 | 4-6 周 | 韩国无线 |
| FDA 510(k) (医疗) | 美国 | $50,000 - $200,000 | 6-12 月 | ❌ 一票否决级 |
| FDA Class II (健康监测) | 美国 | $30,000 - $80,000 | 4-8 月 | ⚠️ 高门槛 |

**典型组合（DTC 出海最低门槛）**：FCC + CE + RoHS + BQB ≈ **\$11,000 - \$25,000** 为基线。

> **打分映射**（L4c）：
> - 总认证 ≤ \$5k → 6 分（无无线 / 仅 SDoC）
> - ≤ \$15k → 4 分（标准 BLE 出海包）
> - ≤ \$40k → 2 分（多国 + Wi-Fi）
> - > \$40k → 0 分（FDA 类，**禁止入场**）

## 4. App / SaaS 运行成本 (USD per MAU per month)

| 场景 | 关键成本 | 单 MAU 月成本 | 备注 |
|---|---|---|---|
| 无 App（蓝牙直连） | - | $0 | 最优 |
| 简单 App（配置/固件升级） | 推送 + OTA | $0.02 - $0.10 | Firebase 即可 |
| 中等 App（数据同步 + 云图表） | 后端存储 + API | $0.10 - $0.40 | 健康/运动数据 |
| 高级 App（AI 转写/翻译） | OpenAI Whisper API + GPT | $0.80 - $3.00 | 录音笔重灾区 |
| 高级 App（自托管 LLM） | GPU 推理 (vLLM) | $0.30 - $1.20 | 自建可降本 |
| 视频/图像云分析 | 计算 + 存储 | $1.50 - $5.00 | ⚠️ 极重 |

> **打分映射**（L4d）：
> - 不需要 App → 5 分
> - SaaS ≤ \$0.20/MAU → 4 分
> - SaaS ≤ \$1.00/MAU → 2 分
> - SaaS > \$1.00/MAU → 0 分
